Terminology

Internet

The internet is the network that keeps all the computers connected allowing computers to communicate between them.

World Wide Web

The World Wide Web is a space in the internet where web related documents are stored. To find this documents you tipically use a URL which are usually linked by hypertext.

Hypertext

The hypertext is the text that is displayed on the computers and other electronic devices.

CLI

CLI or Command Line interface, is something found in every computer. CLI works command based, user inputs a command and the system will proceed to respond.

GUI

The GUI or Graphic User Interface, is a way to show the user information in a graphical way, making it easier to understand.

HCI

The HCI or Human Computer Interaction, is the way humans interact with the computer. An example of this is the mouse.

FTP

The FTP or File Transfer Protocol, is a way to transfer files between machines through a network.

SSH

The SSH or Secure Shell, is a encryoted way to access network services in a secure form over a unsecure network.

HTTP

The Hypertext Transfer Protocol, is the way of communication of the World Wide Web. The HTTP describes how the communication between the web browser and the server.

Telnet

Telnet is a command to get a remote connection to another computer.

HTML

HTML or Hypertext Markup Language, is the code that is used to display web pages. HTML accepts CSS and JavaScript.

CSS

CSS or Cascading Style Sheetsis the type of syntax that is used in HTML to determine the website style, colors, etc. CSS is usually attached to the HTML file externally, but it can also be written internally inside the HTML file.

Javascript

Javascript is a coding language that allows you to create dynamic contents and also allows you to control how the web page acts. web page acts.
